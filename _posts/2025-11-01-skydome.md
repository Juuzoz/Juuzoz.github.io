### Info
During my time at Skydome Entertainment, I mainly worked on their upcoming RPG, Abyssian.
Abyssian is a 2.5D JRPG built in Unity. It has turn-based combat, and contains complex story-telling with decisions that matter.

---

### My Role & Key Contributions
I worked as a programmer on this project, with my main task being the implementation of a quest system.

* The quest system required integration with the existing systems in the game *(Dialogue, Cutscenes)* as well as PixelCrushers tools that the project uses

* Quest creation needed to be possible without coding knowledge

* Wrote a 10+ page manual for documentation

* Created a main and a side quest for the game, so that future devs would have templates to use.

* Implemented the backend for the journal UI and quest HUD

In addition to the quest system, I worked on Character physics, completed numerous QoL and bug fixes, and added Steam Achievements to the game.

* **Engine:** Unity
* **Languages:** C#, Lua.
